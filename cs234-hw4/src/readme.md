# CS 234W - Book App with React
## Author: River
### Date: 02/11/2026

**Description:** CS 234W HW4 | building a react project for the book app

> [!NOTE]
> List of Components Created:
> 1. _App.jsx_ is the main container component which will import all the other components
> 2. _PageHeader.jsx_ is the component with the overview of this project
> 3. _Book.jsx_ is the component for an individual book object
> 4. _BookList.jsx_ is a component made up of individual Book components and will import the _Book_ component to access the book object
> 5. _AddBookForm.jsx_ is a component that allows users to add a new book element

> [!IMPORTANT]
> Instructions for how to download/install/test the project:
> 1. To run react project open the terminal and type _npm run dev_
> 2. A link will appear with a localhost
> 3. Click on the link provided in the terminal and it will take you to your default browser
> 4. The main page will show all the components listed above.
> 5. The form will not work as of now, check back in next week!
